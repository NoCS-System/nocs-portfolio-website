PocketSOS

緊急時対応・多言語サポートアプリ

React Native Expo TypeScript Location

アプリ概要

PocketSOSは、緊急時に必要な情報を素早く表示し、多言語でのコミュニケーションをサポートするアプリです。オフラインで動作し、位置情報の表示、緊急カードの管理、SOS表示機能を提供します。

主な機能

多言語定型文

緊急時に必要な定型文を多言語で表示・音声再生

緊急カード

個人情報や緊急連絡先をカード形式で管理・表示

位置情報表示

現在地の座標をリアルタイムで表示

SOS表示

緊急時のSOS信号を画面に大きく表示

資料管理

PDFや画像などの重要資料を端末内に保存・表示

フラッシュライト

カメラのフラッシュライトを点滅制御

技術仕様

プラットフォーム: iOS / Android
開発環境: React Native + Expo
言語: TypeScript
データ保存: 端末内ローカルストレージ
ネットワーク: オフライン動作

プライバシー・セキュリティ

データは端末内にのみ保存
外部サーバーとの通信なし
個人情報の第三者提供なし
